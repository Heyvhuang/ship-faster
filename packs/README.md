# Packs

Packs are copy-ready modules you can copy into your own project. Each pack includes `COPY_PACK.md` plus a `copy-pack/` folder containing the actual files.

## Available packs

| Pack | Description | Entry |
| --- | --- | --- |
| `2026-01-10-create-saas-starter-pack` | Supabase + Stripe + Credits + R2 | [`2026-01-10-create-saas-starter-pack/`](2026-01-10-create-saas-starter-pack/) |

## Usage

1. Enter a pack directory.
2. Read `COPY_PACK.md`.
3. Copy files from `copy-pack/` into your project and adjust import paths.

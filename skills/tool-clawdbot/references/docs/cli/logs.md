# `clawdbot logs`

Tail Gateway file logs over RPC (works in remote mode).

Related:

* Logging overview: [Logging](/logging)

## Examples

```bash  theme={null}
clawdbot logs
clawdbot logs --follow
clawdbot logs --json
clawdbot logs --limit 500
```

# `clawdbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

* Pairing flow: [Pairing](/start/pairing)

## Commands

```bash  theme={null}
clawdbot pairing list whatsapp
clawdbot pairing approve whatsapp <code> --notify
```

# `clawdbot tui`

Open the terminal UI connected to the Gateway.

Related:

* TUI guide: [TUI](/tui)

## Examples

```bash  theme={null}
clawdbot tui
clawdbot tui --url ws://127.0.0.1:18789 --token <token>
clawdbot tui --session main --deliver
```

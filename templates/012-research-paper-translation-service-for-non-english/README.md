## Quickstart

**Prerequisites:** Node.js 20+, npm

1. Enter template directory:
   ```bash
   cd templates/012-research-paper-translation-service-for-non-english
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000`

## Scripts

## Quickstart

**Prerequisites:** Node.js 20+, npm

1. Enter template directory:
   ```bash
   cd templates/023-llm-data-exfiltration-testing-service
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000`

## Scripts

| Command | Description |
|---------|-------------|

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript

## Quickstart

**Prerequisites:** Node.js 20+, npm

1. Enter template directory:
   ```bash
   cd templates/051-product-launch-calendar-aggregator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000`

## Scripts

| Command | Description |
|---------|-------------|

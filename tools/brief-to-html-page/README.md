# Brief to HTML Page

A small utility pattern for turning a demand brief into a simple static HTML validation page.

This is a tool pattern, not a full runnable starter template.

## Input

```text
Audience:
Problem:
Offer:
Evidence:
CTA:
```

## Output

A single static HTML page that can be reviewed, hosted, or discarded quickly.

## Guardrail

Do not include private prompts, customer data, tokens, or local-only URLs in generated pages.
